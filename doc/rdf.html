rdf

La fonction ~rdf() est une fonction de liste.

Elle prend comme arguments:

  • l'url où se trouve le fichier à parser
  • l'indice à partir duquel afficher les informations
  • le nombre à afficher,
  • les sélecteurs de liste

Les sélecteurs peuvent être: LM, LF, LL, LD ou LE.

Un sélecteur supplémentaire appelé "charset" permet de spécifier le jeu de caractère du flux.

Il peut prendre les valeurs:

  • US-ASCII
  • UTF-8
  • ISO-8859-1

Ce paramètre permet de résoudre des situations qui sont normalement excpetionnelles de nos jours.

Exemple :

~rdf("https://hg.templeet.org/hg/templeet4.0/rss-log",0,10,
  'LF',
    <![LF[
<a href="~rdf_fld('channellink')">~rdf_fld('channeltitle')</a><br/>
<br/>

]LF]>,    
  'LM',
    <![RSS[<a href="~rdf_fld('link')">~rdf_fld('title')</a><br/>
 <br/>
]RSS]>
)
