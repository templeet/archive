Principe général

Dans chaque répertoire de template se trouve un répertoire nommé "lang". Dans ce répertoire se trouvent se trouvent des fichiers sous la forme: <label>.<langue>.tmpl. Le label permet d'identifier un ensemble de traductions correspondant à la même chose. Typiquement ce label aura le même nom que le template qui l'appelle. La langue est un code de deux lettres conforme au code ISO 639-1.
